Overview

Bind a uuid (as a String) to a Resource (a key and a model) The key is a FixNum to find the corresponding resource in the store and the model is the real class of the object (or at least something allowing to find it)

def self.bignum_to_string(b)
  l = Form.inject { |m, n| m+n }
  expand("%0.*x" % [l,b])
end

def self.expand(s)
  match = SplitRegexp.match(s)
  raise InvalidUuidError.new(s) unless match
  match.captures.join('-')
end
