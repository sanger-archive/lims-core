- (Fixnum?) update(object, id, *params)

Doesn't update. At the moment, samples are managed as an external table For testing convenience saving a new object is allowed but update is not. We don't need to modify sample, so when called this method should do nothing. Ideally the external table should be a read-only view.

    the object
