Overview

This Action moves physically tubes from source racks to target racks. It takes an array, which contains the elements of the movement. An element has a source, source_location, target and target_location. Source and targets are tube racks. The source/target_location is the tube location (like "A1") from move the tube and to move the tube.

def _call_in_session(session)
  targets = []
  moves.each do |move|
    source = move["source"]
    from = move["source_location"]
    target = move["target"]
    to = move["target_location"]

    unless source[from].nil?
      target[to] = source[from]
      source[from] = nil
    end

    targets << target
  end

  targets.uniq
end
